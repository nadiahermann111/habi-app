# Habi - Aplikacja do śledzenia nawyków

Aplikacja do śledzenia nawyków z wirtualną małpką, która za wykonane zadania daje monety.

## Technologie
- Backend: Python Flask + SQLite
- Frontend: React
- Baza danych: SQLite3

## Instalacja

### Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
